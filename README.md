# rtn-gimbal-airship-adapter

A React Native adapter for Gimbal and Urban Airship.

## Installation

```sh
npm install rtn-gimbal-airship-adapter
```

## Usage

```js
import { multiply } from 'rtn-gimbal-airship-adapter';

// ...

const result = await multiply(3, 7);
```
